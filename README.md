# fullstack-jwt-auth
тестирую jwt авторизацию

по видео
https://www.youtube.com/watch?v=fN25fMQZ2v0

окружение:
node - 14.18.3
npm - 6.4.15

для запуска прокта нужно:
1. Дозаполнить переменные в файле server/.env SMTP_USER и SMTP_PASSWORD - это логин и пароль от почты Яндекс для рассылки писем
2. Запустить бек - в папке server выполнить `npm run dev`
3. Запустить фронт в соседyем терменале в папке client `npm run start`
